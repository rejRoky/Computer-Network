4. SMTP - 25 
5. DNS - 53 
6. DHCP - 67,68
7. HTTP - 80
8. POP3 - 110
9. NetBIOS Name Service - 137
10. IMAP4 (Internet Message Access Protocol) - 143
11. HTTPS (Secure HTTP) - 443
12. Microsoft-DS (Active Directory) - 445
